# CineScope React Project

CineScope is a movie website built using ReactJS and CSS.

The website includes:
- Movie search
- Category filter
- Movie details pages
- Movie ratings
- Trending movies section
- Favorite button
- Responsive design

Technologies used:
- ReactJS
- CSS3
- Vite
- React Router

How to run the project:

1. Open the project folder in VS Code.
2. Open terminal.
3. Run:

npm install

4. Then run:

npm run dev
